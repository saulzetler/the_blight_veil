# Dungeons & Dragons Campaign Source of Truth

This workspace is designed for managing and editing a markdown file as the central source of truth for your D&D campaign.

## Structure
- `campaign.md`: Main markdown file for campaign notes, lore, NPCs, locations, and session logs.
- `notes/`: Folder for additional markdown files (side quests, character sheets, etc.)
- `.github/copilot-instructions.md`: Workspace instructions for Copilot.
- `README.md`: Overview and usage instructions.

## Recommended Extensions
- Markdown All in One
- Prettier - Code formatter
- Todo Tree (optional for tracking campaign tasks)

## Getting Started
1. Edit `campaign.md` to begin documenting your campaign.
2. Organize supporting notes in the `notes/` folder.
3. Use VS Code's markdown preview for live editing.

Feel free to customize the structure as your campaign grows!
