# **🎨 Optional Style Variants (Mix & Match)**

Use these interchangeable clauses depending on the style you want:

### **Old Archive Style**

> Faded ink, cracked parchment edges, uneven line weight, subtle water stains.

### **Rebel-Drawn Style**

> Rough, uneven ink strokes, hurried sketching, patchy shading, exaggerated landmarks.

### **Concord Military Survey Style**

> Geometric tree icons, precise path lines, crisp borders, minimal decorative elements.

### **High-Fantasy Illustrated Style**

> Slightly more dramatic shading, elegant iconography, gentle glow around magical areas.

# **🏞️ Flexible Components You Can Swap**

Each bracket below is a plug-and-play component:

- **[Perspective]**: top-down / isometric
    
- **[Core region name]**: Ashroot Forest, Rootspire Town, Godscar Ridge, Orrenval outskirts
    
- **[Landmark 1–4]**: roads, ruins, giant trees, cliffs, camps, towers, boundaries
    
- **[Primary Feature]**: one river / one road / coastline / canyon

- **[Shading Style]**: soft hatching / stipple shading / faint wash
    
- **[Ink Style]**: clean lines / rough scribbles / geometric / ornate
